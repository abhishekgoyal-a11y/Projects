Place four square PNG icons here:

  icon16.png   — 16×16 px
  icon32.png   — 32×32 px
  icon48.png   — 48×48 px
  icon128.png  — 128×128 px

Any square PNGs work. Chrome will log warnings if the files are missing,
but the extension loads and runs fine without them.

Quick option: export the same 128×128 image at all four sizes — Chrome
scales it automatically for display.
